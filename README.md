📊 End-to-End Sales Analytics Dashboard (Power BI)

🚀 Project Summary
This project showcases an interactive Sales Analytics Dashboard (2020–2022) built using Power BI.
It converts raw sales data into meaningful business insights using data modeling, DAX, and visualizations.

📌 Objectives
* Analyze sales, cost, and profit
* Track product and brand performance
* Monitor order status (Order → Shipped → Delivered)
* Compare state-wise and supervisor-wise performance

🗂️ Dataset

The dataset includes:
* Order ID
* Product
* Category & Brand
* State & Supervisor
* Sales, Cost, Profit
* Quantity
* Order Status

🧩 Data Modeling
* Created relationships between Sales, Supervisor, and State tables
* Used Order ID as primary key
* Relationship flow: Supervisor → State → Sales

📊 Key Features
* Product-wise sales analysis
* Category & brand performance
* Order tracking
* State-wise insights
* Supervisor comparison

📈 KPIs

* Total Sales
* Total Cost
* Total Profit
* Quantity Sold
* Transactions

🛠️ Tools Used
* Power BI
* DAX
* Data Modeling
* Data Visualization

💡 Learnings
* Data modeling & relationships
* DAX calculations
* Dashboard creation
* Business insights generation
